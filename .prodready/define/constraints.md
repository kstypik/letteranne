# Constraints

## Deployment
- Target: VPS (Docker) with Caddy as reverse proxy
- Region: To be decided

## Scale
- Launch: ~200 users
- 6 months: ~5,000 users

## Budget
- Infrastructure: Free tier only for now
- Tooling: Prefer free/open-source tools

## Compliance & Security
- GDPR compliance required

## Tech Stack Preferences
- Language: Python (backend), TypeScript strict mode (frontend)
- Framework:
  - Backend: Django headless API using django-modern-rest (not DRF, not Django Ninja), django-allauth in headless mode, uv for package manager, ruff for lint/format
  - Frontend: React SPA with TanStack libraries (Query/Form/Router preferred), pnpm for package manager,  oxilint for linter, oxfmt for formatter
- Database: PostgreSQL
- ORM: Django ORM (built in)
- Additional:
  - Async jobs: Django 6.0 built-in task queue (DB backend)
  - API contracts: OpenAPI generated by backend and consumed by frontend via Orval
  - Testing: pytest + pytest-django (backend), Vitest + MSW (frontend), Playwright (e2e)
