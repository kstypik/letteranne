# Vision

## Problem Statement
Current instant messaging pushes fast, shallow communication and leaves many people exhausted. This app offers intentional, delayed digital letters so conversations can be slower, deeper, and more meaningful.

## Target Users
People of all ages who are tired of instant messaging: younger users looking for thoughtful peer connections and older users seeking a retro post-mail feeling in a digital format.

## Core Value Proposition
Deliver an exceptional typing and writing experience that helps users craft beautiful letters and recreates the feeling of classical letter communication.

## Success Metrics
- Average letter length
- Day-30 retention

## MVP Scope

### Must Have (MVP)
- Sign up/login
- Sending letters
- Letters history
- Sending open letters (everyone can see and reply to)
- Finding users by ID (friends)
- Finding users (simple random list for MVP)
- Postcards collection (achievements and events)
- Sending a postcard in a letter (from user's collection)
- User profile (bio, avatar, etc.)

### Nice to Have (Future)
- Advanced search for penpals
- Interests/hobbies settings with matching impact
- E-shop for postcards
- Letter drafts
- Buying postcards for others
- Reporting users
- Blocking users
- Temporarily hiding users
- Personal notes about users
- Location sharing
- Matching by location
- Reusable paragraph library for frequent letter content
- Advanced privacy settings
- Excluding people based on interests/location
- Gender/age preferences
- Languages spoken
- Writing habits metadata (response speed, typical letter length)
